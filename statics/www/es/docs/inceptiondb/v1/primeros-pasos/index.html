Guía de Inicio Rápido con InceptionDB

Esta guía te ayudará a empezar con InceptionDB mediante la creación de una colección, la inserción de tres elementos y la consulta de elementos filtrados.

Paso 1: Crear una Colección

Para crear una colección en InceptionDB, necesitas definir el nombre de la colección. Aquí hay un ejemplo de cómo hacerlo usando curl en bash:

1curl -X POST "https://example.com/v1/databases/tu-id-de-base-de-datos/collections" \
2-H "Api-Key: tu-api-key" \
3-H "Api-Secret: tu-api-secret" \
4-d '{
5"name": "mi-colección"
6}'

Respuesta Esperada

1{
2	"defaults": {
3		"id": "uuid()"
4	},
5	"indexes": 0,
6	"name": "mi-colección",
7	"total": 0
8}

Paso 2: Insertar Elementos

Una vez que hayas creado la colección, puedes insertar elementos en ella. A continuación se muestra cómo insertar tres elementos en la colección recién creada:

1curl -X POST "https://example.com/v1/databases/tu-id-de-base-de-datos/collections/mi-colección/documents" \
2-H "Api-Key: tu-api-key" \
3-H "Api-Secret: tu-api-secret" \
4-d '{"nombre": "Elemento 1", "valor": 100}
5{"nombre": "Elemento 2", "valor": 200}
6{"nombre": "Elemento 3", "valor": 300}
7'

Paso 3: Listar Elementos con Filtro

Para listar elementos en la colección con un filtro específico, puedes usar el siguiente ejemplo de solicitud con curl:

1curl -X POST "https://example.com/v1/databases/tu-id-de-base-de-datos/collections/mi-colección/find" \
2-H "Api-Key: tu-api-key" \
3-H "Api-Secret: tu-api-secret" \
4-d '{
5      "filter": {
6        "valor": { "$gte": 200 }
7      }
8    }'

Respuesta Esperada

 1{
 2		"id": "id-del-documento-2",
 3		"nombre": "Elemento 2",
 4		"valor": 200
 5}
 6{
 7		"id": "id-del-documento-3",
 8		"nombre": "Elemento 3",
 9		"valor": 300
10}

Resumen

Siguiendo estos pasos, has creado una colección en InceptionDB, insertado tres elementos y consultado elementos con un filtro. Esto te proporciona una base sólida para empezar a trabajar con InceptionDB y aprovechar sus capacidades para gestionar tus datos de manera eficiente.
