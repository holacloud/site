GET /schedulers/{id}/health

Devuelve el estado de salud de un scheduler.

Autenticación

Este endpoint es público. No requiere autenticación.

Parámetros de Ruta

Parámetro Tipo Descripción
id string Identificador único del scheduler

Ejemplo de Solicitud

1curl "https://api.hola.cloud/schedulers/sched-a1b2c3d4-e5f6-7890-abcd-ef1234567890/health"

Ejemplo de Respuesta

1HTTP/1.1 200 OK
2Content-Type: application/json

1{
2  "status": "ok",
3  "uptime_seconds": 12345
4}

Códigos de Error

Estado Código Descripción
404 not_found Scheduler no encontrado
500 internal_error Error interno del servidor
