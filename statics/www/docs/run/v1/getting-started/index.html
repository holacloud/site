Getting Started with Run

Run is a container execution service with a built-in Docker Registry v2 compatible API. It allows you to deploy, run, and manage container images with environment variables and volumes.

Authentication

Run uses DockerAuth with Basic Auth for registry authentication. Use your HolaCloud API key and secret as the username and password.

Logging into the Registry

1docker login https://registry.hola.cloud

You will be prompted for your HolaCloud API key (username) and API secret (password).

Building and Pushing an Image

Create a simple application with a 

1Dockerfile
:

1FROM alpine:latest
2COPY app.sh /app.sh
3RUN chmod +x /app.sh
4CMD ["/app.sh"]

Build and tag the image:

1docker build -t registry.hola.cloud/my-project/my-app:v1 .

Push to the registry:

1docker push registry.hola.cloud/my-project/my-app:v1

Starting a Container

Start a container from a pushed image via the Console API:

 1curl -X POST "https://api.hola.cloud/api/console/run" \
 2  -H "Api-Key: YOUR_API_KEY" \
 3  -H "Api-Secret: YOUR_API_SECRET" \
 4  -H "Content-Type: application/json" \
 5  -d '{
 6    "image": "registry.hola.cloud/my-project/my-app:v1",
 7    "env": {
 8      "LOG_LEVEL": "info",
 9      "DATABASE_URL": "postgres://..."
10    },
11    "volumes": [
12      {
13        "container_path": "/data",
14        "volume_name": "my-data"
15      }
16    ]
17  }'

Expected response:

1{
2  "id": "c8a9f3b2-1234-5678-9abc-def012345678",
3  "image": "registry.hola.cloud/my-project/my-app:v1",
4  "status": "running",
5  "created_at": "2025-06-21T12:00:00Z"
6}

Stopping a Running Container

Stop a container by its ID:

1curl -X DELETE "https://api.hola.cloud/api/console/run/c8a9f3b2-1234-5678-9abc-def012345678" \
2  -H "Api-Key: YOUR_API_KEY" \
3  -H "Api-Secret: YOUR_API_SECRET"

Rolling Back to a Previous Image Version

Deploy a previous image version by specifying a different tag:

1curl -X PUT "https://api.hola.cloud/api/console/run/c8a9f3b2-1234-5678-9abc-def012345678" \
2  -H "Api-Key: YOUR_API_KEY" \
3  -H "Api-Secret: YOUR_API_SECRET" \
4  -H "Content-Type: application/json" \
5  -d '{
6    "image": "registry.hola.cloud/my-project/my-app:v0"
7  }'

Managing Environment Variables

Update environment variables for a running container:

 1curl -X PUT "https://api.hola.cloud/api/console/env" \
 2  -H "Api-Key: YOUR_API_KEY" \
 3  -H "Api-Secret: YOUR_API_SECRET" \
 4  -H "Content-Type: application/json" \
 5  -d '{
 6    "env": {
 7      "LOG_LEVEL": "debug",
 8      "DATABASE_URL": "postgres://..."
 9    }
10  }'

Managing Volumes

Attach volumes to a container:

 1curl -X PUT "https://api.hola.cloud/api/console/volumes" \
 2  -H "Api-Key: YOUR_API_KEY" \
 3  -H "Api-Secret: YOUR_API_SECRET" \
 4  -H "Content-Type: application/json" \
 5  -d '{
 6    "volumes": [
 7      {
 8        "container_path": "/data",
 9        "volume_name": "my-data",
10        "read_only": false
11      }
12    ]
13  }'

Checking Running State

Query the current state of all containers:

1curl "https://api.hola.cloud/api/console/run" \
2  -H "Api-Key: YOUR_API_KEY" \
3  -H "Api-Secret: YOUR_API_SECRET"

Expected response:

 1[
 2  {
 3    "id": "c8a9f3b2-1234-5678-9abc-def012345678",
 4    "image": "registry.hola.cloud/my-project/my-app:v1",
 5    "status": "running",
 6    "env": { "LOG_LEVEL": "info" },
 7    "volumes": [{ "container_path": "/data", "volume_name": "my-data" }],
 8    "created_at": "2025-06-21T12:00:00Z"
 9  }
10]

Complete Workflow Example

A full end-to-end workflow:

  1. Login to the registry:

    1docker login https://registry.hola.cloud

  2. Build your image:

    1docker build -t registry.hola.cloud/my-project/my-app:v1 .

  3. Push the image:

    1docker push registry.hola.cloud/my-project/my-app:v1

  4. Start a container:

    1curl -X POST "https://api.hola.cloud/api/console/run" \
2  -H "Api-Key: YOUR_API_KEY" \
3  -H "Api-Secret: YOUR_API_SECRET" \
4  -H "Content-Type: application/json" \
5  -d '{"image": "registry.hola.cloud/my-project/my-app:v1"}'

  5. Verify it is running:

    1curl "https://api.hola.cloud/api/console/run" \
2  -H "Api-Key: YOUR_API_KEY" \
3  -H "Api-Secret: YOUR_API_SECRET"

  6. Stop the container when done:

    1curl -X DELETE "https://api.hola.cloud/api/console/run/CONTAINER_ID" \
2  -H "Api-Key: YOUR_API_KEY" \
3  -H "Api-Secret: YOUR_API_SECRET"
