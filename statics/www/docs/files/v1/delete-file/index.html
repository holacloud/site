DELETE /v1/buckets/{id}/files/*

Delete a file from a bucket. The file path is specified after 

1/files/
in the URL.

Authentication

Requires 

1Api-Key
and 
1Api-Secret
 headers.

Path Parameters

Parameter Type Description 
1id
 string The bucket ID (e.g., 
1bkt_abc123
)

Request

1curl -X DELETE "https://api.hola.cloud/v1/buckets/bkt_abc123/files/images/logo.png" \
2  -H "Api-Key: YOUR_API_KEY" \
3  -H "Api-Secret: YOUR_API_SECRET"

Response

1HTTP/1.1 204 No Content

Error Codes

Status Code Description
401 Unauthorized Missing or invalid API credentials
404 Not Found The specified bucket or file does not exist
500 Internal Server Error An unexpected error occurred
