Getting Started

This guide walks you through creating a queue, writing messages, reading them via long-poll, checking queue stats, and cleaning up.

Prerequisites

  • A HolaCloud account.
  • curl installed on your machine.

Step 1: Create a Queue

Create a new queue by sending a POST request with a name:

1curl -X POST "https://api.hola.cloud/v1/queues" \
2  -H "Content-Type: application/json" \
3  -d '{
4    "name": "my-queue"
5  }'

Expected response:

1{
2  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
3  "name": "my-queue",
4  "length": 0,
5  "reads": 0,
6  "writes": 0
7}

Save the returned id — you will use it in subsequent requests.

Step 2: Write Messages

Tailon accepts a stream of JSON objects, one per line (NDJSON). Write three messages:

1curl -X POST "https://api.hola.cloud/v1/queues/a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
2  -H "Content-Type: application/x-ndjson" \
3  -d '{"type": "greeting", "text": "Hello"}
4{"type": "greeting", "text": "Hola"}
5{"type": "command", "action": "ping"}'

Expected response:

1{
2  "written": 3
3}

Step 3: Read Messages

Use a long-poll GET to retrieve messages. The Limit header controls how many messages to return:

1curl -X GET "https://api.hola.cloud/v1/queues/a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
2  -H "Limit: 2"

Expected response:

 1[
 2  {
 3    "id": "m1",
 4    "type": "greeting",
 5    "text": "Hello",
 6    "timestamp": "2025-06-21T12:00:01Z"
 7  },
 8  {
 9    "id": "m2",
10    "type": "greeting",
11    "text": "Hola",
12    "timestamp": "2025-06-21T12:00:01Z"
13  }
14]

Messages are returned in FIFO order. Repeat the request to retrieve the remaining messages.

Step 4: Check Queue Stats

Inspect the queue details, including its current length and total reads/writes:

1curl "https://api.hola.cloud/v1/queues/a1b2c3d4-e5f6-7890-abcd-ef1234567890"

Expected response:

1{
2  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
3  "name": "my-queue",
4  "length": 1,
5  "reads": 2,
6  "writes": 3
7}

Step 5: Delete the Queue

When you are done, delete the queue to free resources:

1curl -X DELETE "https://api.hola.cloud/v1/queues/a1b2c3d4-e5f6-7890-abcd-ef1234567890"

Expected response: HTTP 204 No Content.
