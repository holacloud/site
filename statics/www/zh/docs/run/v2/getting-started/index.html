Run快速入门

Run 是一个容器执行服务，内置 Docker Registry v2 兼容 API。您可以使用它部署、运行和管理带有环境变量和卷的容器镜像。

身份认证

Run 使用 DockerAuth 配合基本认证（Basic Auth）进行注册表身份验证。使用您的 HolaCloud API 密钥和 API 密钥 Secret 作为用户名和密码。

登录到注册表

1docker login https://registry.hola.cloud

系统将提示您输入 HolaCloud API 密钥（用户名）和 API 密钥 Secret（密码）。

构建和推送镜像

创建一个简单的应用程序和 

1Dockerfile
：

1FROM alpine:latest
2COPY app.sh /app.sh
3RUN chmod +x /app.sh
4CMD ["/app.sh"]

构建并标记镜像：

1docker build -t registry.hola.cloud/my-project/my-app:v1 .

推送到注册表：

1docker push registry.hola.cloud/my-project/my-app:v1

启动容器

通过 Console API 从已推送的镜像启动容器：

 1curl -X POST "https://api.hola.cloud/api/console/run" \
 2  -H "Api-Key: 你的API密钥" \
 3  -H "Api-Secret: 你的API密钥Secret" \
 4  -H "Content-Type: application/json" \
 5  -d '{
 6    "image": "registry.hola.cloud/my-project/my-app:v1",
 7    "env": {
 8      "LOG_LEVEL": "info",
 9      "DATABASE_URL": "postgres://..."
10    },
11    "volumes": [
12      {
13        "container_path": "/data",
14        "volume_name": "my-data"
15      }
16    ]
17  }'

预期响应：

1{
2  "id": "c8a9f3b2-1234-5678-9abc-def012345678",
3  "image": "registry.hola.cloud/my-project/my-app:v1",
4  "status": "running",
5  "created_at": "2025-06-21T12:00:00Z"
6}

停止正在运行的容器

通过容器 ID 停止容器：

1curl -X DELETE "https://api.hola.cloud/api/console/run/c8a9f3b2-1234-5678-9abc-def012345678" \
2  -H "Api-Key: 你的API密钥" \
3  -H "Api-Secret: 你的API密钥Secret"

回滚到之前的镜像版本

通过指定不同的标签来部署之前的镜像版本：

1curl -X PUT "https://api.hola.cloud/api/console/run/c8a9f3b2-1234-5678-9abc-def012345678" \
2  -H "Api-Key: 你的API密钥" \
3  -H "Api-Secret: 你的API密钥Secret" \
4  -H "Content-Type: application/json" \
5  -d '{
6    "image": "registry.hola.cloud/my-project/my-app:v0"
7  }'

管理环境变量

更新运行中容器的环境变量：

 1curl -X PUT "https://api.hola.cloud/api/console/env" \
 2  -H "Api-Key: 你的API密钥" \
 3  -H "Api-Secret: 你的API密钥Secret" \
 4  -H "Content-Type: application/json" \
 5  -d '{
 6    "env": {
 7      "LOG_LEVEL": "debug",
 8      "DATABASE_URL": "postgres://..."
 9    }
10  }'

管理卷

将卷挂载到容器：

 1curl -X PUT "https://api.hola.cloud/api/console/volumes" \
 2  -H "Api-Key: 你的API密钥" \
 3  -H "Api-Secret: 你的API密钥Secret" \
 4  -H "Content-Type: application/json" \
 5  -d '{
 6    "volumes": [
 7      {
 8        "container_path": "/data",
 9        "volume_name": "my-data",
10        "read_only": false
11      }
12    ]
13  }'

查询所有容器的当前状态：

1curl "https://api.hola.cloud/api/console/run" \

完整工作流程示例

完整的端到端工作流程：

  1. 登录到注册表：

    1docker login https://registry.hola.cloud

  2. 构建你的镜像：

    1docker build -t registry.hola.cloud/my-project/my-app:v1 .

  3. 推送镜像：

    1docker push registry.hola.cloud/my-project/my-app:v1

  4. 启动容器：

    1curl -X POST "https://api.hola.cloud/api/console/run" \
2  -H "Api-Key: 你的API密钥" \
3  -H "Api-Secret: 你的API密钥Secret" \
4  -H "Content-Type: application/json" \
5  -d '{"image": "registry.hola.cloud/my-project/my-app:v1"}'

  5. 验证它正在运行：

    1curl "https://api.hola.cloud/api/console/run" \
2  -H "Api-Key: 你的API密钥" \
3  -H "Api-Secret: 你的API密钥Secret"

  6. 停止容器：

    1curl -X DELETE "https://api.hola.cloud/api/console/run/CONTAINER_ID" \
2  -H "Api-Key: 你的API密钥" \
3  -H "Api-Secret: 你的API密钥Secret"
