使用Holamail发送邮件

Holamail 是一个轻量级的 SMTP 服务器，用于发送事务性电子邮件。它实现了标准的 SMTP 协议（HELO/EHLO、MAIL FROM、RCPT TO、DATA、QUIT），监听 TCP 端口 2525，无需 TLS。

连接信息

参数 值
主机 smtp.hola.cloud
端口 2525
加密 无（纯 SMTP）

SMTP 协议流程

原始 SMTP 对话遵循以下顺序：

1HELO client.example.com
2MAIL FROM:<noreply@holacloud.app>
3RCPT TO:<user@example.com>
4DATA
5Subject: 来自Holamail的问候
6
7这是邮件正文。
8.
9QUIT

示例

使用 curl

curl 原生支持 SMTP：

1curl --mail-from noreply@holacloud.app \
2     --mail-rcpt user@example.com \
3     --upload-file email.txt \
4     smtp://smtp.hola.cloud:2525

其中 email.txt 包含：

1Subject: 来自Holamail的问候
2
3这是邮件正文。

使用 openssl s_client

通过纯 TCP 连接手动进行 SMTP 通信：

 1exec 3<>/dev/tcp/smtp.hola.cloud/2525
 2echo "HELO client.example.com" >&3
 3echo "MAIL FROM:<noreply@holacloud.app>" >&3
 4echo "RCPT TO:<user@example.com>" >&3
 5echo "DATA" >&3
 6echo "Subject: 你好" >&3
 7echo "" >&3
 8echo "来自Holamail的问候！" >&3
 9echo "." >&3
10echo "QUIT" >&3
11cat <&3

使用 Go 的 net/smtp

 1package main
 2
 3import (
 4    "log"
 5    "net/smtp"
 6)
 7
 8func main() {
 9    to := []string{"user@example.com"}
10    msg := []byte("To: user@example.com\r\n" +
11        "Subject: 来自Holamail的问候\r\n" +
12        "\r\n" +
13        "这是邮件正文。\r\n")
14
15    err := smtp.SendMail("smtp.hola.cloud:2525", nil,
16        "noreply@holacloud.app", to, msg)
17    if err != nil {
18        log.Fatal(err)
19    }
20}

使用 Python 的 smtplib

 1import smtplib
 2from email.message import EmailMessage
 3
 4msg = EmailMessage()
 5msg.set_content("这是邮件正文。")
 6msg["Subject"] = "来自Holamail的问候"
 7msg["From"] = "noreply@holacloud.app"

 9
10with smtplib.SMTP("smtp.hola.cloud", 2525) as server:

使用场景

  • 密码重置 — 按需发送安全的重置链接。
  • 通知 — 发送警报、提醒和状态更新。
  • 系统告警 — 从监控系统（Nagios、Prometheus、Grafana）触发邮件。
  • Lambda 集成 — 从 Lambda 函数调用 Holamail 以编程方式发送邮件：
 1import smtplib
 2from email.message import EmailMessage
 3
 4def handler(event, context):
 5    msg = EmailMessage()
 6    msg.set_content(event.get("body", ""))
 7    msg["Subject"] = event.get("subject", "通知")
 8    msg["From"] = "noreply@holacloud.app"
 9    msg["To"] = event["to"]
10
11    with smtplib.SMTP("smtp.hola.cloud", 2525) as server:
12        server.send_message(msg)
13
14    return {"statusCode": 200, "body": "邮件已发送"}

错误处理

Holamail 返回标准 SMTP 状态码：

状态码 含义
250 请求完成
550 邮箱不可用（收件人无效）
551 收件人被拒绝
552 超出存储配额
553 地址无效
554 事务失败

当收件人地址无效时，服务器返回 550。请在你的应用程序代码中处理连接错误（超时、拒绝连接）——这些通常表示网络问题或服务暂时不可用。
