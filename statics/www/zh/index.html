品牌配色

HolaCloud 的视觉语言

以下配色定义了我们在所有产品中的品牌形象，这些颜色作为 CSS 自定义变量发布在 /css/brand.css 中，便于统一使用。

以深邃的蓝宝石为基调，辅以亮丽的蔚蓝与薄荷色，传递自信与创新；沙色提供温暖背景，Slate 与 Ink 负责文字呈现。

蓝宝石 Hex #102347 变量 --hola-color-sapphire
蔚蓝 Hex #1F6FEB 变量 --hola-color-cerulean
薄荷 Hex #3DD6C2 变量 --hola-color-mint
沙色 Hex #F5E7D0 变量 --hola-color-sand
石板 Hex #1F2937 变量 --hola-color-slate
云白 Hex #F4F7FB 变量 --hola-color-cloud
