📁 Files

Servicio seguro de almacenamiento de objetos

Files ofrece una API REST para buckets y para subir, descargar, listar, consultar metadatos y eliminar archivos.

Comenzar Documentación
Files

Funcionalidades

Organiza archivos en buckets y accede a ellos mediante endpoints HTTP simples.

API de buckets

Crea, lista, consulta y elimina buckets con nombre y descripción.

Operaciones de archivos

Sube, descarga, lista, inspecciona y elimina archivos dentro de buckets.

Metadatos almacenados

La API devuelve tamaño, tipo MIME, timestamps y hashes de contenido.

Quick start

Inicio rápido

Empieza con Files en segundos usando la API.

curl -X POST https://files.hola.cloud/v1/buckets \
  -H 'X-Glue-Authentication: {"user":{"id":"user-123"}}' \
  -d '{"name":"my-bucket","description":"Uploads for my app"}'
