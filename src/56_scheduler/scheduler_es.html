⏰ Scheduler

Cola de tareas diferidas con leases

Scheduler gestiona tareas diferidas, reserva con leases, ack de tareas y snapshots por HTTP.

Comenzar Documentación
Scheduler

Funcionalidades

Crea namespaces de scheduler, encola tareas diferidas, reserva trabajo, extiende leases y confirma finalización.

Tareas diferidas

Encola payloads JSON con IDs explícitos y duraciones estilo Go como 30s o 5m.

Workers con leases

Reserva tareas por una ventana de trabajo, extiende leases y luego confirma o reprograma tareas.

Stream de snapshots

Observa tareas programadas y en curso mediante respuestas snapshot y snapshots SSE.

Quick start

Inicio rápido

Get started with Scheduler in seconds using the API.

curl -X POST https://scheduler.hola.cloud/schedulers \
  -d '{"id":"jobs","display_name":"Background jobs"}' \
  && curl -X POST https://scheduler.hola.cloud/schedulers/jobs/tasks \
  -d '{"id":"task-001","delay":"60s","payload":{"job":"backup"},"labels":["kind:backup"]}'
