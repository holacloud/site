Quick start

快速开始

Get started with Scheduler in seconds using the API.

curl -X POST https://api.hola.cloud/v1/schedulers \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"name": "nightly-backup", "schedule": "0 2 * * *", "action": {"type": "webhook", "url": "https://myapp.com/backup"}}'
