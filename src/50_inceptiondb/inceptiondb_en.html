Quick start

Quick start

Get started with InceptionDB in seconds using the API.

curl -X POST https://api.hola.cloud/v1/databases \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"name": "mydb"}'

# Create a collection
curl -X POST https://api.hola.cloud/v1/databases/mydb/collections \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"name": "users"}'

# Insert a document
curl -X POST https://api.hola.cloud/v1/databases/mydb/collections/users \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"name": "Alice", "email": "alice@example.com"}'
