🗄️ InceptionDB

Base de datos NoSQL administrada

InceptionDB almacena documentos JSON en bases de datos y colecciones expuestas mediante una API HTTP.

InceptionDB

Funcionalidades

Crea bases de datos, gestiona colecciones y trabaja con documentos JSON usando endpoints basados en acciones.

Modelo de documentos flexible

Guarda documentos JSON con esquemas dinámicos en colecciones y consúltalos con filtros e índices.

Operaciones de colección

Usa acciones de colección como :insert, :find, :patch y :remove.

Claves de base de datos

Accede a bases de datos mediante autenticación de owner o claves API de base de datos.

Quick start

Inicio rápido

Empieza con InceptionDB en segundos usando la API.

curl -X POST https://inceptiondb.hola.cloud/v1/databases \
  -H 'X-Glue-Authentication: {"user":{"id":"user-123"}}' \
  -d '{"name": "mydb"}'

# Crea una colección
curl -X POST https://inceptiondb.hola.cloud/v1/databases/database-123/collections \
  -H 'Api-Key: $API_KEY' -H 'Api-Secret: $API_SECRET' \
  -d '{"name": "users"}'

# Inserta un documento
curl -X POST https://inceptiondb.hola.cloud/v1/databases/database-123/collections/users:insert \
  -H 'Api-Key: $API_KEY' -H 'Api-Secret: $API_SECRET' \
  -d '{"name": "Alice", "email": "alice@example.com"}'
