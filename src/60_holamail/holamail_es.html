Holamail

Funcionalidades

Usa Holamail como endpoint SMTP ligero mientras evoluciona el producto de correo de producción.

Listener SMTP

Acepta comandos SMTP básicos como HELO/EHLO, MAIL FROM, RCPT TO, DATA y QUIT.

Captura de mensajes

Recibe datos de mensaje y los registra en logs en lugar de entregar correo a destinatarios externos.

Útil para desarrollo

Útil para pruebas locales e integraciones que necesitan un endpoint compatible con SMTP.

Quick start

Inicio rápido

Envía un mensaje de prueba con cualquier cliente SMTP.

swaks --server smtp.testmail.hola.cloud:25 \
  --from sender@example.com \
  --to user@example.com \
  --header 'Subject: Hello from Holamail' \
  --body 'This is a test message.'
