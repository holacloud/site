Quick start

Inicio rápido

Get started with Run in seconds using the API.

# Sube una imagen
curl -X POST https://api.hola.cloud/v1/run/push \
  -H "Authorization: Bearer $TOKEN" \
  -F "image=@myapp.tar.gz" \
  -F "name=myapp:latest"

# Ejecuta un contenedor
curl -X POST https://api.hola.cloud/v1/run/exec \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"image": "myapp:latest", "port": 8080, "scale": 2}'
