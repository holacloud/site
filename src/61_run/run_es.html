🚀 Run

Ejecución de contenedores con registro Docker integrado

Run combina un subconjunto Docker Registry v2 orientado a push con endpoints de consola para arrancar, detener, hacer rollback y configurar repositorios.

Comenzar Documentación
Run

Funcionalidades

Sube imágenes de contenedor al registro integrado y gestiona el runtime del repositorio con la API de consola.

Runtime por repositorio

Arranca, detiene y revierte el runtime asociado a un repositorio.

Entorno y volúmenes

Guarda variables de entorno y definiciones de volúmenes con la API de consola.

Registro orientado a push

Usa los endpoints Docker Registry v2 implementados para subir capas y manifests.

Quick start

Inicio rápido

Get started with Run in seconds using the API.

# Consulta datos de consola del repositorio
curl 'https://run.hola.cloud/api/console?repository=my-project/my-app'

# Arranca el runtime del repositorio
curl -X POST https://run.hola.cloud/api/console/start \
  -d '{"repository":"my-project/my-app","reference":"latest"}'
