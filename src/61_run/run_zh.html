🚀 Run

容器执行服务，内置 Docker 注册表

Run 将面向 push 的 Docker Registry v2 子集与 console endpoints 结合，用于启动、停止、回滚和配置 repository。

开始使用 文档中心
Run

核心功能

将容器镜像推送到内置 registry，并通过 console API 管理 repository runtime 状态。

Repository Runtime

启动、停止和回滚与 repository 关联的 runtime。

环境和卷

通过 console API 保存环境变量和卷定义。

面向 Push 的 Registry

使用已实现的 Docker Registry v2 上传 endpoints 推送镜像层和 manifests。

Quick start

快速开始

Get started with Run in seconds using the API.

# 查看 repository console data
curl 'https://run.hola.cloud/api/console?repository=my-project/my-app'

# 启动 repository runtime
curl -X POST https://run.hola.cloud/api/console/start \
  -d '{"repository":"my-project/my-app","reference":"latest"}'
