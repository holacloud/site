⚙️ Config

集中化配置管理

Config 存储用户级 JSON 配置条目，并暴露 v0 和 v1 HTTP API。

开始使用 文档中心
Config

核心功能

读取和更新配置 map，而无需为应用额外构建配置服务。

用户配置

使用 GET /v1/config 和 PATCH /v1/config 管理认证用户的配置。

Entries Map

将配置存储为 JSON entries，包括字符串、数字、布尔值和嵌套对象。

Legacy v0 配置

需要时可管理带有 id 和 entries 字段的 v0 配置文档。

Quick start

快速开始

使用 API 在几秒内开始使用 Config。

curl -X PATCH https://config.hola.cloud/v1/config \
  -H 'X-Glue-Authentication: {"user":{"id":"user-123"}}' \
  -d '{"database.url":"postgres://prod:5432/db","feature.enabled":true}'
