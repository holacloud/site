# Holamail API 参考

Holamail 是一个基于 **SMTP 的电子邮件服务** — 它不提供 HTTP API。客户端使用标准 SMTP 协议通过端口 2525 连接。

## SMTP 参考

有关 SMTP 命令、连接示例和最佳实践的完整指南，请参阅 [SMTP 协议参考](./smtp-reference)。

## 连接详情

| 设置 | 值 |
|---------|-------|
| 主机 | `smtp.hola.cloud` |
| 端口 | `2525` |
| 协议 | SMTP（无需 TLS） |
| 认证 | 无（为已验证项目开放中继） |

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| SMTP | `2525` | 用于发送电子邮件的原始 SMTP 连接 |
