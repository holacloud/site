# Documentación

<b><a href="/docs/inceptiondb/">InceptionDB</a></b>: Tu base de datos optimizada para rendimiento y escalabilidad. Administra y consulta tus datos con eficiencia.

<b><a href="/docs/lambda/">Lambda System</a></b>: Sistema de ejecución flexible para tus aplicaciones. Escala automáticamente sin gestionar servidores.

<b><a href="/docs/files/">Files</a></b>: Almacenamiento de objetos seguro y accesible. Guarda, recupera y organiza tus archivos con facilidad.

<b><a href="/docs/config/">Config</a></b>: Sistema de configuración centralizado. Gestiona la configuración de tus aplicaciones en un solo lugar.

<b><a href="/docs/logs/">InstantLogs</a></b>: Sistema de logs en tiempo real. Monitoriza y analiza la actividad de tus aplicaciones instantáneamente.

<b><a href="/docs/queues/">Tailon</a></b>: Sistema de colas robusto para manejar cargas de trabajo intensivas y operaciones asíncronas.

<b><a href="/docs/scheduler/">Scheduler</a></b>: Programador de tareas distribuido para ejecución de trabajos temporizados.

<b><a href="/docs/kvnode/">KVNode</a></b>: Almacén clave-valor replicado con backends de persistencia configurables.

<b><a href="/docs/console/">Consola</a></b>: Consola de gestión web para todos los servicios de HolaCloud.

<b><a href="/docs/glue2/">API Gateway (Glue2)</a></b>: Capa central de autenticación y enrutamiento para todos los servicios.

<b><a href="/docs/holamail/">Holamail</a></b>: Servicio SMTP para comunicaciones transaccionales.

<b><a href="/docs/run/">Run</a></b>: Servicio de ejecución de contenedores con registro Docker integrado.
