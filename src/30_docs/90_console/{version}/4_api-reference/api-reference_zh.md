# 控制台 API 参考

控制台公开了一组模拟 API 端点，用于在开发和演示模式下的 Web 界面。这些端点模拟后端服务，无需身份验证。

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| `GET` | `/version` | 返回当前控制台版本 |
| `GET` | `/fakeapi/*` | 用于开发的模拟 API 端点 |

### GET /version

返回控制台服务的版本字符串。

### GET /fakeapi/*

通配符端点，为各种后端服务提供模拟数据，包括身份验证、InceptionDB、Lambda、Files 和项目管理。
