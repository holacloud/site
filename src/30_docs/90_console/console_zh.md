# 控制台

控制台是 HolaCloud 的基于 Web 的管理界面。它提供了一个统一的平台来管理所有 HolaCloud 服务——数据库、函数、文件、配置等。

![控制台仪表板](img/console.png)

## 功能特性

### 仪表板
一目了然地监控项目的整体运行状况。仪表板显示服务状态、近期活动以及每个活跃服务的关键指标。

### 数据库管理
直接在浏览器中浏览、查询和管理 InceptionDB 数据库。创建集合、检查文档、执行临时查询，无需离开控制台。

### Lambda 管理
查看、创建、编辑和调用 Lambda 函数。控制台包含内置代码编辑器和调用日志查看器，用于调试无服务器函数。

### 文件浏览器
上传、下载和组织 Files 服务中的文件。浏览器支持目录导航、文件预览和批量操作。

### 配置编辑器
通过结构化编辑器管理应用程序配置。单独编辑键、以 JSON 格式导入/导出配置，并跟踪变更历史。

### 日志查看器
实时流式传输和搜索 InstantLogs 的日志。按严重级别、服务或时间范围过滤，快速定位问题。

### 项目设置
配置当前控制台 UI 公开的项目设置。

## 访问方式

身份验证后，可通过项目的 HolaCloud 仪表板 URL 访问控制台。详细设置和使用说明可在应用程序内部找到。
