# Console

The Console is the web-based management UI for HolaCloud. It provides a unified interface for managing all HolaCloud services — databases, functions, files, configuration, and more.

![Console Dashboard](img/console.png)

## Features

### Dashboard
Monitor the overall health of your project at a glance. The dashboard shows service status and key metrics for each active service.

### Database Management
Browse, query, and manage InceptionDB databases directly from the browser. Create collections, inspect documents, and run ad-hoc queries without leaving the Console.

### Lambda Management
View, create, edit, and invoke Lambda functions. The Console includes a built-in code editor and invocation log viewer for debugging serverless functions.

### File Browser
Upload, download, and organize files in the Files service. The browser supports directory navigation, file preview, and bulk operations.

### Config Editor
Manage application configuration through a structured editor. Edit individual keys, import/export configuration as JSON, and track changes over time.

### Log Viewer
Stream and search logs from InstantLogs in real time. Filter by severity, service, or time range to quickly pinpoint issues.

### Project Settings
Configure project-wide settings exposed by the current Console UI.

## Availability

The Console is accessible at your project's HolaCloud dashboard URL after authentication. Detailed setup and usage instructions are available within the application itself.
