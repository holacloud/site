# Consola

La Consola es la interfaz de administración web de HolaCloud. Proporciona un entorno unificado para gestionar todos los servicios de HolaCloud — bases de datos, funciones, archivos, configuración y más.

![Panel de la Consola](img/console.png)

## Funcionalidades

### Panel Principal
Monitorea la salud general de tu proyecto de un vistazo. El panel muestra el estado de los servicios y las métricas clave de cada servicio activo.

### Administración de Bases de Datos
Explora, consulta y administra bases de datos InceptionDB directamente desde el navegador. Crea colecciones, inspecciona documentos y ejecuta consultas ad-hoc sin salir de la Consola.

### Administración de Lambda
Visualiza, crea, edita e invoca funciones Lambda. La Consola incluye un editor de código integrado y un visor de registros de invocación para depurar funciones serverless.

### Explorador de Archivos
Sube, descarga y organiza archivos en el servicio Files. El explorador admite navegación por directorios, vista previa de archivos y operaciones por lotes.

### Editor de Configuración
Administra la configuración de la aplicación mediante un editor estructurado. Edita claves individualmente, importa/exporta configuración como JSON y realiza seguimiento de cambios.

### Visor de Registros
Transmite y busca registros de InstantLogs en tiempo real. Filtra por severidad, servicio o rango de tiempo para identificar rápidamente problemas.

### Configuración del Proyecto
Configura los parámetros generales del proyecto expuestos por la interfaz actual de la Consola.

## Disponibilidad

La Consola es accesible desde la URL del panel de control de tu proyecto en HolaCloud después de la autenticación. Las instrucciones detalladas de uso están disponibles dentro de la propia aplicación.
