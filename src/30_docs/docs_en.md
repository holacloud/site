# Documentation

<b><a href="/docs/inceptiondb/">InceptionDB</a></b>: Your database optimized for performance and scalability. Manage and query your data efficiently.

<b><a href="/docs/lambda/">Lambda System</a></b>: Flexible execution system for your applications. Scale automatically without managing servers.

<b><a href="/docs/files/">Files</a></b>: Secure and accessible object storage. Save, retrieve, and organize your files with ease.

<b><a href="/docs/config/">Config</a></b>: Centralized configuration system. Manage the configuration of your applications in one place.

<b><a href="/docs/logs/">InstantLogs</a></b>: Real-time log system. Instantly monitor and analyze the activity of your applications.

<b><a href="/docs/queues/">Tailon</a></b>: Robust queue system to handle intensive workloads and asynchronous operations.

<b><a href="/docs/scheduler/">Scheduler</a></b>: Distributed task scheduler for time-based and delayed job execution.

<b><a href="/docs/kvnode/">KVNode</a></b>: Replicated key-value store with configurable persistence backends.

<b><a href="/docs/console/">Console</a></b>: Web-based management console for all HolaCloud services.

<b><a href="/docs/glue2/">API Gateway (Glue2)</a></b>: Central authentication and routing layer for all services.

<b><a href="/docs/holamail/">Holamail</a></b>: SMTP email service for transactional communications.

<b><a href="/docs/run/">Run</a></b>: Container execution service with built-in Docker registry.
