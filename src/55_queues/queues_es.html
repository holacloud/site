📨 Tailon

Sistema de colas de mensajes y trabajos asíncronos

Tailon es una API ligera de colas JSON con creación de colas, escrituras, lecturas y estadísticas simples.

Tailon

Funcionalidades

Usa Tailon cuando necesites una cola HTTP sencilla para mensajes JSON.

Colas con nombre

Crea colas por nombre y consulta longitud, lecturas y escrituras actuales.

Escrituras JSON

Escribe payloads JSON directamente en una cola con el endpoint :write.

Lecturas bloqueantes

Lee mensajes desde el endpoint :read como JSON delimitado por saltos de línea.

Quick start

Inicio rápido

Empieza con Tailon en segundos usando la API.

curl -X POST https://queues.hola.cloud/v1/queues \
  -d '{"name":"orders"}' \
  && curl -X POST https://queues.hola.cloud/v1/queues/orders:write \
  -d '{"message":"order-12345"}'
