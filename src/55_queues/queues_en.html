Key features

Reliable messaging

Pub/Sub patterns

Create fan-out topologies for event-driven architectures.

Consumer groups

Quick start

Quick start

Get started with Tailon in seconds using the API.

curl -X POST https://api.hola.cloud/v1/queues \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"name": "orders", "type": "fifo"}' \
  && curl -X POST https://api.hola.cloud/v1/queues/orders \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"message": "order-12345"}'
