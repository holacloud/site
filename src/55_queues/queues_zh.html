📨 Tailon

消息队列与异步工作负载系统

Tailon 是轻量级 JSON 消息队列 API，支持创建队列、写入、读取和查看简单统计。

开始使用 文档中心
Tailon

核心功能

当你需要简单的 HTTP JSON 消息队列时，可以使用 Tailon。

命名队列

按名称创建队列，并查看当前长度、读取次数和写入次数。

JSON 写入

通过 :write endpoint 将 JSON payload 直接写入队列。

阻塞读取

从 :read endpoint 读取换行分隔的 JSON 消息。

Quick start

快速开始

使用 API 在几秒内开始使用 Tailon。

curl -X POST https://queues.hola.cloud/v1/queues \
  -d '{"name":"orders"}' \
  && curl -X POST https://queues.hola.cloud/v1/queues/orders:write \
  -d '{"message":"order-12345"}'
