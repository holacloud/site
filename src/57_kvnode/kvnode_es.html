🔑 KVNode

Almacén clave-valor replicado de alto rendimiento

KVNode expone colecciones y claves por HTTP, con sincronización root/replica mediante su protocolo WAL.

Comenzar Documentación
KVNode

Funcionalidades

Guarda valores por colección y clave, lista claves y replica cambios entre nodos.

Colecciones y claves

Crea colecciones y establece, obtiene, lista o elimina claves con la API HTTP v1.

Modelo root/replica

Las réplicas siguen a un nodo root usando los comandos de protocolo implementados por KVNode.

Escrituras versionadas

Las operaciones de escritura devuelven información de secuencia y versión para sincronización.

Quick start

Inicio rápido

Empieza con KVNode en segundos usando la API.

curl -X POST https://kvnode.hola.cloud/v1/collections \
  -H 'Api-Key: $API_KEY' -H 'Api-Secret: $API_SECRET' \
  -d '{"collection":"users"}' \
  && curl -X POST https://kvnode.hola.cloud/v1/collections/users/keys/123 \
  -H 'Api-Key: $API_KEY' -H 'Api-Secret: $API_SECRET' \
  -d '{"value":{"name":"Alice"}}'
