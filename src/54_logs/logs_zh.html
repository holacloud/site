Quick start

快速开始

Get started with InstantLogs in seconds using the API.

curl -X POST https://logs.hola.cloud/v1/loggers \
  -H 'X-Glue-Authentication: {"user":{"id":"user-123"}}' \
  -d '{"name":"app-logger"}' \
  && printf 'info Hello world\n' | curl https://logs.hola.cloud/v1/loggers/logger-123/ingest \
  -H 'Api-Key: $LOGGER_KEY' -H 'Api-Secret: $LOGGER_SECRET' \
  --data-binary @-
