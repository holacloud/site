Quick start

Inicio rápido

Get started with InstantLogs in seconds using the API.

curl -X POST https://api.hola.cloud/v1/loggers \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"name": "app-logger", "retention_days": 30}' \
  && curl https://api.hola.cloud/v1/loggers/my-logger/ingest \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"level":"info","message":"Hello world"}'
