🔗 Glue2

API Gateway and authentication layer

Glue2 is HolaCloud's host-based reverse proxy and service glue layer.

Get started Documentation
Glue2

Key features

Route virtual hosts, inject HolaCloud authentication headers, and inspect gateway status and traffic stats.

Virtual hosts

List the virtual hosts currently known by the gateway.

Authentication context

Inject JSON authentication context through X-Glue-Authentication when a project requires it.

Gateway diagnostics

Expose version, virtual hosts, project host status, OpenAPI, and traffic statistics endpoints.

Quick start

Quick start

Get started with Glue2 in seconds using the API.

curl https://glue2.hola.cloud/version
curl https://glue2.hola.cloud/v0/virtualhosts
curl https://glue2.hola.cloud/v0/stats
